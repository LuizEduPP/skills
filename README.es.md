# AI Skills Collection

![AI Skills Collection](cover-skill.png)

> 🌐 Idiomas: [English](README.md) | **Español** | [Português (BR)](README.pt-BR.md)

Una colección de skills portátiles para IA - templates, guías y automatizaciones que funcionan con cualquier asistente de IA.

## ¿Qué son las Skills?

Las skills son módulos de conocimiento reutilizables que potencian el desarrollo asistido por IA. Cada skill contiene:
- Documentación especializada (SKILL.md)
- Scripts de automatización
- Templates y referencias
- Flujos de trabajo estandarizados

### Dónde funcionan

Las skills son **módulos de conocimiento portátiles** que funcionan en cualquier lugar:

- ✅ Cualquier asistente de IA que lea archivos markdown
- ✅ Claude Code, Claude Desktop, Claude Web
- ✅ Copilot Chat, Cursor, Windsurf, Gemini CLI
- ✅ Referencia manual — solo lee los archivos SKILL.md

El formato `SKILL.md` es markdown plano con frontmatter — usable en cualquier lugar.

## Cómo Usar

### Cargar una Skill

En asistentes de IA como Claude Code, Windsurf y Cursor, las skills se cargan automáticamente según la descripción de la tarea. También puedes mencionarlo explícitamente:

```
Use skill [nombre-de-skill] para [tarea]
```

### Crear una Nueva Skill

1. Crea un directorio con el nombre de la skill
2. Agrega un `SKILL.md` con frontmatter:

```markdown
---
name: mi-skill
description: Qué hace esta skill. Usar cuando [casos de uso].
---

# Título de la Skill

Contenido e instrucciones...
```

3. Agrega scripts, templates y referencias según sea necesario
