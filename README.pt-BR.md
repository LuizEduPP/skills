# AI Skills Collection

![AI Skills Collection](cover-skill.png)

> 🌐 Idiomas: [English](README.md) | [Español](README.es.md) | **Português (BR)**

Uma coleção de skills portáteis para IA - templates, guias e automações que funcionam com qualquer assistente de IA.

## O que são Skills

Skills são módulos de conhecimento reutilizáveis que potencializam o desenvolvimento assistido por IA. Cada skill contém:
- Documentação especializada (SKILL.md)
- Scripts de automação
- Templates e referências
- Workflows padronizados

### Onde funcionam

Skills são **módulos de conhecimento portáteis** que funcionam em qualquer lugar:

- ✅ Qualquer assistente de IA que leia arquivos markdown
- ✅ Claude Code, Claude Desktop, Claude Web
- ✅ Copilot Chat, Cursor, Windsurf, Gemini CLI
- ✅ Referência manual — basta ler os arquivos SKILL.md

O formato `SKILL.md` é markdown puro com frontmatter — utilizável em qualquer lugar.

## Como Usar

### Carregando uma Skill

Em assistentes de IA como Claude Code, Windsurf e Cursor, as skills são carregadas automaticamente com base na descrição da tarefa. Você também pode mencionar explicitamente:

```
Use skill [nome-da-skill] para [tarefa]
```

### Criando uma Nova Skill

1. Crie um diretório com o nome da skill
2. Adicione um `SKILL.md` com frontmatter:

```markdown
---
name: minha-skill
description: O que esta skill faz. Use quando [casos de uso].
---

# Título da Skill

Conteúdo e instruções...
```

3. Adicione scripts, templates e referências conforme necessário
